## mlog

the mlog is multi-level logger based on go std log. It is:

* Simple
* Easy to use
* Easy to extend a new logger like color log, syslog, rolling file log ...

NOTHING ELSE

```
package main

import (
    log "github.com/ccpaging/log/mlog"
)

func main() {
    log.Debug("This is Debug")
    log.Info("This is Info")
}
```
